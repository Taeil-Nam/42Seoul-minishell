# 42Seoul-minishell
## 목표
- 나만의 쉘을 만들어보자.

## 관련 개념
- Lexical Analisys.
- Tokenization.
- bash 쉘 동작 과정.
- System Call
- 멀티 프로세스
- 프로세스 계층 구조.
- Redirection
- Pipe(IPC)
- File Descriptor

## 개발 및 테스트 환경
- Intel CPU 기반 Mac.
- MacOS Version = 12.6.2

## 사용 방법
- make = 쉘 실행 파일 생성.
- 실행 방법
  - ./minishell
